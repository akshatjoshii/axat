UseState - Maintain States in Functional Component

How to use states inside a functional component in React?.

Akshat Â  Follow

A functional component is a reusable, self-contained code block that divides the user interface into smaller pieces. It is basically a JavaScript function that returns a JSX (HTML look-alike React element). It looks like this:

const App = (props) => {
  return (
    <div>This is just an example</div>
    );
 }

React offers a special function (Hooks) that enables developers to use React features inside simple looking functional components. Here, we will use useState which is a Hook that offers us to use React state to function components.

# Declaring State

React provides a named export useState which can be imported like this:

import React, { useState } from 'react';

Now, inside a functional component we can use useState like this:

import React, { useState } from 'react';

function Post() {
  // Declare a new state variable, which we'll call "content"
  const [content, setContent] = useState('Default Text');

useState here declares a new state variable called content, instead of content, it can be anything like person, place, item, user or whatever. The argument passed to useState('Default Text') is the initial state of the state variable. We can pass anything to it, number, string, boolean etc depending on the requirement. In our example we passed a string as the initial state of the state variable content, which we want to change on a button's click. useState returns a pair of values. One is the current state (ie, content) and the other is a function (setContent) to update the corresponding state. To change the state of content

It is important to note that the names in the left content and setContent are not the part of React API. We can name however we want, for example:

  const [person, setPerson] = useState('John');

# Reading State Variables

Displaying state variable is simple. All we need to do is:

  <p>{content}</p>

# Updating State Variable

To update the state variable, we pass new value as the argument to setContent function like this:

<button onClick={() => setContent('This is new text')}>
    Change Content
</button>

# Conclusion

The complete code will somewhat look like this:

  import React, { useState } from 'react';
 
   function Post() {
     const [content, setContent] = useState('Default Text');
     return (
       <div>
         <p>You clicked {count} times</p>
         <button onClick={() => setContent('This is new text')}>
             Change Content
         </button>
      </div>
    );
  }

We first imported the useState Hook from React which enables us to keep local state in a function component. Inside Post component, we declared a new state variable by calling the useState Hook, which returns a pair of values, to which we give names. We are calling our variable as content and initialize it with a string passed as an argument to useState call. When the user clicks the button, setContent is called with a new value. React will then re-render the Post component, passing the new content value to it.
